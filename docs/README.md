# SonarFC 文档导航

## 必读文档

- [SonarFC-PRD.md](./SonarFC-PRD.md)
  产品目标、用户、范围、功能模块、商业模式和阶段排期。

- [SonarFC-Technical-Design.md](./SonarFC-Technical-Design.md)
  技术架构、数据模型、数据库 Schema、成本估算和周末验证实现方案。

- [SonarFC-UI-Design.md](./SonarFC-UI-Design.md)
  配色系统、页面结构、组件风格和视觉方向。

## 参考原型

- [sonarfc-prototype.jsx](./sonarfc-prototype.jsx)
  现有高保真原型，适合复用信息结构和视觉语言。

- [sonarfc-prototype.html](./sonarfc-prototype.html)
  可直接在浏览器打开的静态预览页，无需构建环境，支持切换比赛卡查看页面效果。

## 推荐阅读顺序

1. 先读 PRD，确认产品边界和优先级。
2. 再读 UI 设计文档，确认颜色和页面层级。
3. 再读技术设计文档，确认实现路径和数据约束。
4. 先打开 HTML 预览页看交互和整体效果。
5. 最后对照 JSX 原型，判断哪些 UI 可以直接迁移，哪些需要重构。
